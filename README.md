# GO！むらっぴ Version 1.6.4

Version 1.6.3を基礎に、ステージ編集から配置できる3種類のアイテムを追加した差分パッチです。

この成果物には `assets` フォルダーを含めていません。現在使用している画像・音声を維持したまま、プログラム部分だけを更新できます。

## 今回追加したアイテム

| アイテム | 効果 | 標準画像 |
|---|---|---|
| ハート | 体力を1回復。体力上限は3 | `assets/items/item_heart.png` |
| 残機UP | 残機を1増加。上限は99 | `assets/items/item_life.png` |
| 空中ジャンプ強化 | 10秒間、空中ジャンプが1回から2回へ増加。地上ジャンプを含めて合計3回ジャンプ可能 | `assets/items/item_air_jump.png` |

ステージ編集画面の「配置するパターン」に、以下が追加されます。

- アイテム：ハート
- アイテム：残機UP
- アイテム：空中ジャンプ強化

同じアイテムを1ステージ内へ複数配置できます。取得済みアイテムは、そのステージで残機を失って復帰しても再出現しません。ステージを最初から開始し直すと再出現します。

## アイテム画像が未配置の場合

画像ファイルが存在しなくてもゲームは停止しません。ゲーム画面とステージ編集画面では、次の簡易表示を自動的に使用します。

| アイテム | 代替表示 |
|---|---|
| ハート | ハート型 |
| 残機UP | `1UP` |
| 空中ジャンプ強化 | `J+` |

画像を準備した後、指定ファイル名で配置すれば自動的に置き換わります。

## アイテム効果音

配置先は `assets/audio/` です。

| タイミング | 標準ファイル名 |
|---|---|
| ハート取得 | `item_heart.mp3` |
| 残機UP取得 | `item_life.mp3` |
| 空中ジャンプ強化取得 | `item_air_jump.mp3` |

MP3が存在しない、または読み込めない場合は、内蔵のWeb Audio効果音へ自動的に切り替わります。

## 空中ジャンプ強化の仕様

通常状態では以下です。

```text
地上ジャンプ 1回
＋
空中ジャンプ 1回
＝ 合計2回
```

強化中は以下になります。

```text
地上ジャンプ 1回
＋
空中ジャンプ 2回
＝ 合計3回
```

標準の有効時間は10秒です。画面上部のステージ表示に `JUMP+ 10s` のように残り時間を表示します。

設定は `js/config.js` で変更できます。

```javascript
airJumpBoostMaxJumps: 3,
airJumpBoostDuration: 10.0,
```

## パッチ適用方法

現在GitHub Pagesへ配置しているゲーム一式へ、次の3ファイルを上書きしてください。

```text
js/config.js
js/game.js
service-worker.js
```

`assets` フォルダーは上書きしません。

アイテム専用画像を使用する場合は、ご自身で次のフォルダーとファイルを追加してください。

```text
assets/
└─ items/
   ├─ item_heart.png
   ├─ item_life.png
   └─ item_air_jump.png
```

アイテム効果音を使用する場合は、次を追加してください。

```text
assets/
└─ audio/
   ├─ item_heart.mp3
   ├─ item_life.mp3
   └─ item_air_jump.mp3
```

## GitHub Pages更新時の注意

Version 1.6.4では、画像・音声・HTML・JavaScriptについて、オンライン時はGitHub Pages上の最新版を優先するService Workerへ変更しています。

それでも旧表示が残る場合は、次の順に実施してください。

1. GitHubのActionsでPagesの公開完了を確認
2. AndroidのChromeで公開URLを開く
3. ページを2回再読み込み
4. ホーム画面版アプリを完全終了して再起動

サイトデータ削除は、編集マップやハイスコアも消える可能性があるため最終手段です。実施前に「マップ一式を書き出し」を行ってください。

## 既存データの引き継ぎ

保存キーはVersion 1.5以降と同じため、同じGitHub Pages URLで起動した場合、次を引き継ぎます。

- 編集済みステージ
- 追加したマップパターン
- ハイスコア
- 端末内ランキング
- サウンド設定

既存ステージへアイテムを配置するには、更新後にステージ編集画面を開いて追加してください。
