# GO！むらっぴ 可変ジャンプ仕様 Version 1.6.5

## 基本動作

画面を押した瞬間にジャンプを開始します。
指を早く離すと上昇速度を早めに弱め、長く押すほど従来の最大ジャンプへ近づきます。

- ジャンプ力は **レベル1～5**
- レベル1：ごく短いタップ
- レベル5：長押しによる最大ジャンプ
- 1段目と2段ジャンプの両方に適用
- 押している間にジャンプ開始を遅らせる方式ではないため、操作の反応速度は維持

## 標準の判定時間

| レベル | タップ・長押し時間 | 上昇速度の残し方 |
|---|---:|---:|
| 1 | 0～0.06秒 | 32% |
| 2 | 0.06秒超～0.12秒 | 42% |
| 3 | 0.12秒超～0.18秒 | 54% |
| 4 | 0.18秒超～0.24秒 | 69% |
| 5 | 0.24秒超～0.32秒以上 | 100% |

レベル5は、0.32秒以上押し続けてもそれ以上強くなりません。

2段ジャンプは空中で扱いやすいように、短いタップ時の減速を少し弱くしています。

| レベル | 2段ジャンプの上昇速度の残し方 |
|---|---:|
| 1 | 36% |
| 2 | 47% |
| 3 | 59% |
| 4 | 73% |
| 5 | 100% |

## 調整場所

`js/config.js` の `gameplay` 内です。

```javascript
jumpPowerMaxLevel: 5,
jumpHoldMaxTime: 0.32,
jumpPowerLevelTimes: [0.06, 0.12, 0.18, 0.24, 0.32],
jumpReleaseVelocityFactors: [0.32, 0.42, 0.54, 0.69, 1.00],
doubleJumpReleaseVelocityFactors: [0.36, 0.47, 0.59, 0.73, 1.00],
```

### 長押し時間を長くしたい場合

例として最大判定を0.50秒にする場合です。

```javascript
jumpHoldMaxTime: 0.50,
jumpPowerLevelTimes: [0.10, 0.20, 0.30, 0.40, 0.50],
```

ただし、スマートフォンのアクションゲームでは0.50秒はやや長く、操作が重く感じやすいため、標準は0.32秒です。

### 短いタップでも高く跳ばせたい場合

レベル1～4の係数を大きくします。

```javascript
jumpReleaseVelocityFactors: [0.42, 0.56, 0.70, 0.86, 1.00],
```

### 高さの差をさらに大きくしたい場合

レベル1～3の係数を小さくします。

```javascript
jumpReleaseVelocityFactors: [0.24, 0.40, 0.58, 0.80, 1.00],
```

## 入力処理

- Android / iPhone：Pointer Events の押下・解放時間を計測
- Windows：マウス押下・解放時間を計測
- キーボード：Spaceキーまたは上矢印キーの押下・解放時間を計測
- 指を画面外で離した場合や操作がキャンセルされた場合も、解放として処理
- 初回タップで音声機能を解除する場合でも、短いタップを取りこぼさない順序で処理

## 既存仕様との関係

- 2段ジャンプ：維持
- ジャンプバッファ：維持
- コヨーテタイム：維持
- 穴の縁への引っ掛かり：維持
- 敵踏みつけ後の反発：可変ジャンプの対象外
- ダメージによる吹き飛び：可変ジャンプの対象外
